Client Server Computing

by Angela Matta

What is a Client Server Model?

The client/server model is a form of computer infrastructure that allows for users to access servers. The user's computer or software acts as a "client" and requests something from a server(another computer or bit of software). The server then responds to these requests which allow for users to access common things like games, web sites and databases. For example, a user opens a web browser like Fire Fox and then types in google.com into the search bar. The server that hosts that web site is going to respond by sending the infromation for the website to the user so that they can access the site.

Advantages of the client/server model

Several major advantages of the client/server model are:

  1. Model is easily scalable for more computing power to handle more users
  2. Model allows for centralized security,storage,data backup and recovery etc.
  3. Model allows for international communication
  4. Model allows for better managment of users

Examples of TCP/IP protocols

  • FTP: File Transfer Protocal, rules that dictate how computers transfer files between other computers or systems
  • HTTP: HyperText Transfer Protocal,protocal for communicating between web browsers and web servers
  • HTTPS: Hypertext Transfer Protocol Secure, extension of HTTP, adds on security to HTTP
  • DHCP: Dynamic Host Configuration Protocol, network management protocal used for assigning IP addresses and other communications as apart of the client/server model
  • SMTP:Simple Mail Transfer Protocol, standard protocals for sending emails over the web

Examples of network utilities and services

  • ping: allows user to test if host is reachable/valid, also allows user to view IP address for domain name
  • tracert: traces path packet of information takes to a destination(ie. client, other servers etc.), allows for diagnosis of slow/down hosts
  • pathping: combination of ping and tracert
  • whois: allows user to see what domain name is available for use
  • ipconfig: allows for configuration of IP address,either through release,renew,all etc.
